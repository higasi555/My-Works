==================== TomatoFes ====================

TomatoFesとは、スペインの伝統的な祭りであるトマト祭りを参考に作成されたゲームです。
トマトを投げつけて、敵を倒そう！


==================== Requirement ==================

【Software】
・Java-17.0.10
・Openjfx-21.0.1…jreフォルダに同包しています

（いずれもwindows環境の方であれば不要です）


【Hardware】

◆推奨動作要件

CPU：第12世代 Intel Core i5以上 or Apple M1チップ以上
GPU：NVIDIA Geforce GTX1660以上
メモリ：16GB以上

◆最低動作要件（※30fpsでの場合です）

CPU：第8世代 Intel Core i5
GPU：Intel UHD 620
メモリ：8GB

※ゲームの特性上GPU、メモリリソースを多く消費します
==================== Installation =================

==================== Usage ========================

【Windows環境の方】
1. exeファイルをダブルクリック
2. 画面が表示されたら、左上でfps、中央で遊びたいステージを選択し、ゲームスタート

【Linux環境の方】
▼exeが実行できる環境の方
Windows環境と同手順で起動可能です

▼exeが実行できない環境の方

1.javafxのsdkを適当な場所へダウンロード
2.端末上で
java --module-path "%javafxsdkまでのパス%\javafx-sdk-21.0.1\lib" --add-modules javafx.controls,javafx.graphics,javafx.fxml -jar TomatoFes.jar
を実行
3. 画面が表示されたら、左上でfps、中央で遊びたいステージを選択し、ゲームスタート

【Mac環境の方】
m(._.*)m... すみません、当方に動作環境が無いため、検証できていません！
